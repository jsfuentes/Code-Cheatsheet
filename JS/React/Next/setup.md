# Setup

```bash
npx create-next-app

npm init next-app nextjs-blog --example "https://github.com/vercel/next-learn-starter/tree/master/learn-starter"
```

## Usage

```bash
yarn dev
```

## Absolute Path Imports

Jsconfig.json

```js
{
  "compilerOptions": {
    "paths": {
      "src": ["src"]
    }
  }
}
```
