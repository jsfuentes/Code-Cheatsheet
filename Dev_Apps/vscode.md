# Featured Shortcuts

## VSCode Navigation

| Command     | Result                      |
| ----------- | --------------------------- |
| Cmd-Shift-P | Command Palette             |
| Cmd-P       | Open a File                 |
| Cmd-T       | Find class, ft, or variable |
| Cmd-Shift-E | Explorer Open               |
| Cmd-K Cmd-W | Close all open files        |
| Cmd-B       | Toggle SidePanel            |
|             |                             |
|             |                             |
|             |                             |

## Editor Navigation

| Cmd         | Result              |
| ----------- | ------------------- |
| Cmd-L       | Select Current Line |
| Cmd-Shift-K | Delete Current Line |
|             |                     |
