How content should be resized to fit container

| Prop              | Effect                                 |
| ----------------- | -------------------------------------- |
| object-fit: fill; | stretch to fill                        |
| contaiin          | Keep aspect ratio                      |
| Cover             | Cover all of container even if cut off |
