## Old

| Value        | Effect                                                       |
| ------------ | ------------------------------------------------------------ |
| block        | Starts on new line, fill width of parent container           |
| inline       | Remain in line and only take up width of content(height/width have no effect) |
| inline-block | Remain in line, can apply height and width                   |
|              |                                                              |
