# Tmux

tmux is a terminal multiplexer that keeps your sesison running even when your computer loses its connection to the devserver.

## Basics

- new						=>`tmux` - new
- acces previous			=> `tmux attach`

### In tmux

`Ctrl-b` base command 

c  create window
w  list windows
n  next window
p  previous window
f  find window
,  name window
&  kill window
