# AWS

Elastic Beanstalk is a fancy wrapper for EC2 instance with like load-balancing and shit

## EC2 Instance
Create one online at Amazon
Get your key
ssh with your key into EC2
yum install git
git clone your repo
Make sure your server is running on port 80 and go wild
