# Setup

```bash
yarn global add gatsby-cli
```

## Gatsby

```bash
gatsby new my-site
gatsby new my-site [starter link]
gatsby new my-site https://github.com/Vagr9K/gatsby-advanced-starter

cd my-site
gatsby develop
```

## Basic Usage

Add a new page to `pages/a.js` and it will be available at `localhost:8000/a`
